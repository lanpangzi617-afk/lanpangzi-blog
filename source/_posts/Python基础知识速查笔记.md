# Python 基础知识速查

## 1. 基础语法

### 1.1 三元表达式

**作用**

> 用于在一行代码中完成简单的条件判断并返回结果，使代码更加简洁、易读。

**语法**

```python
x if condition else y
```

**示例**

```python
x = 3
y = 5
result = x if x < y else y
print(result)  # 输出 3
```
